# Example Website

This is a simple example website to demonstrate ESP32 hosting capabilities.

## Files

- `index.html` - Main page
- `style.css` - Dark theme stylesheet
- `script.js` - Simple JavaScript functionality

## Usage

1. Open admin panel
2. Connect to ESP32 via Bluetooth
3. Upload these files
4. Click "Start Web Server"
5. Access your website at the ESP32's IP address

## Customization

Replace these files with your own website! The ESP32 can serve:

- HTML files
- CSS stylesheets
- JavaScript
- Images (PNG, JPG, GIF, SVG)
- JSON files
- Any static content

### Tips

- **Minify files** to save space (1.5MB total limit)
- **Optimize images** before uploading
- **Test locally** before uploading to ESP32
- **Keep it simple** - ESP32 has limited processing power

Current example: ~3KB total (plenty of room!)
